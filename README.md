This project was built using these technologies.

- JavaScript
- HTML
- CSS
- BootStrap
- Particles.js
- Deployed on Vercel

## Features are:

- Add item to todo list.
- Edit your existing todo item.
- Mark your todo item as done.
- Delete your todo item from the list.
